# Yifan Yang Digital Portfolio

Portfolio site built with React and Vite

## Tech stack

- React
- Vite
- ESLint

## Run locally

```bash
npm install
npm run dev
```

Then open the URL shown by Vite (typically `http://localhost:5173`).

## Production build

```bash
npm run build
npm run preview
```

The optimized output is generated in `dist/`.
